# endsession checklist

## Session capture
- [ ] Full conversation read from start to finish
- [ ] Skill or task being worked on identified
- [ ] Current state of the work identified
- [ ] All decisions made this session identified
- [ ] All open or unresolved items identified
- [ ] Next action identified — specific and concrete, not vague

## Layer 2
- [ ] All five fields populated: what we worked on, current state, decisions, open items, next action
- [ ] Specific enough that PM can pick up without re-reading the conversation
- [ ] No fabrication — if something is unclear, marked [TBD] not invented

## Layer 1
- [ ] Full conversation scanned for explicit preference statements
- [ ] If found: appended to relevant Layer 1 section, not overwritten
- [ ] If not found: Layer 1 left unchanged

## Output
- [ ] Complete primer.md output includes both Layer 1 and Layer 2
- [ ] Clean markdown, ready to copy-paste
- [ ] Delivery instruction included for Claude.ai users
