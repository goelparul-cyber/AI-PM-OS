# startsession checklist

## Before resuming, confirm:
- [ ] primer.md has been loaded and both layers read fully
- [ ] PM identity and preferences confirmed from Layer 1
- [ ] Active project identified from Layer 2
- [ ] Next action identified from Layer 2
- [ ] Layer 1 preferences noted and will be applied silently throughout session
- [ ] No contradictions between Layer 1 standing decisions and Layer 2 session decisions

## Format checks
- [ ] Resume briefing is 3-4 lines max
- [ ] Next action is prompted, not stated
- [ ] No bullet points in output
- [ ] No meta-references to primer.md or memory system

## If any item fails:
Flag to PM before proceeding. Do not resume without resolving.
